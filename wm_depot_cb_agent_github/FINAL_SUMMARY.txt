================================================================================
  TROUBLESHOOTING
================================================================================

PROBLEM: "Module not found" error
SOLUTION: Check requirements.txt has all dependencies
  uv pip freeze > requirements.txt
  git add requirements.txt
  git push
  Redeploy

PROBLEM: "Authentication failed"
SOLUTION: Check .streamlit/secrets.toml has correct GCP credentials
  1. Verify service account email is correct
  2. Verify BigQuery permissions are set
  3. Re-add secrets in Streamlit Cloud

PROBLEM: "No data returned"
SOLUTION: Check depot number and date range
  1. Try depot 7634 (test depot)
  2. Increase "Days Lookback" to 90
  3. Check BigQuery tables directly

PROBLEM: Streamlit Cloud deployment fails
SOLUTION: Check deployment logs
  1. Click app settings (gear icon)
  2. View "Logs" section
  3. Search for ERROR messages
  4. Read Streamlit documentation for specific error

MORE HELP: See DEPLOYMENT_CHECKLIST.md and README.md
